        *   NXC                                   → `-u [USER] -H [HASH] -d [DOMAIN] [--local-auth]` → Add `-k` if `STATUS_NOT_SUPPORTED` 
        *   Impacket                         → `[DOMAIN]/[USER]@[IP] -hashes [HASH]`
        *   Password Conversion  → `iconv -f ASCII -t UTF-16LE <(printf "[PASS]") | openssl dgst -md4`
        *   Cracking                          → `hashcat -m 1000`
